add_data 

add_data(df: DataFrame) -> None

Ingests raw simulation data and updates the internal data state.

This method appends the provided DataFrame to self.data. Because this changes the underlying dataset, all downstream results (clean_data, sufficiency_results, and pod_results) are reset to empty states.

pod 

pod(poi_col: str, threshold: float, bandwidth_ratio: float = 0.1, n_boot: int = 1000) -> Dict[str, Any]

Runs the full 'Generalized â vs a' pipeline.

  1. Fits Robust Mean Model (Polynomial Selection).
  2. Fits Variance Model (Kernel Smoothing).
  3. Infers Statistical Distribution (AIC Selection).
  4. Calculates PoD Curve.
  5. Calculates 95% Confidence Bounds (Bootstrap).

Parameters:

 required

The detection threshold (e.g., 4.0 dB).

 required
bandwidth_ratio float

Smoothing window size as a fraction of data range (Default 0.1).

 0.1
n_boot int

Number of bootstrap iterations for confidence bounds (Default 1000).

 1000

Returns:

Name Type Description
Dict Dict[str, Any]

A dictionary containing all curves and models needed for plotting/reporting.

Parameters:

Name Type Description Default

 10

validate 

validate() -> None

Cleans and validates the raw data stored in self.data.

This method filters the raw data based on project-specific rules. It populates self.clean_data with valid rows and self.removed_data with invalid ones.

Side Effects

Updates self.clean_data and self.removed_data. Resets self.clean_data to empty if validation fails critically.

infer_best_distribution 

infer_best_distribution(residuals: ndarray, X: ndarray, bandwidth: float) -> Tuple[str, Tuple]

Selects the best statistical distribution for the standardized residuals using AIC.

This function normalizes residuals by their local standard deviation (Z-scores) and tests them against a suite of candidate distributions (Normal, Gumbel, Logistic, Laplace, t-Student).

Parameters:

Name Type Description Default
residuals ndarray

Raw residuals from the mean model.

 required
X ndarray

Input locations for the residuals.

 required
bandwidth float

Bandwidth used for local standardization.

 required

Returns:

Type Description
Tuple[str, Tuple]

Tuple[str, Tuple]: - best_name: The SciPy name of the best-fitting distribution (e.g., 'norm'). - best_params: The fitted parameters for that distribution (e.g., loc, scale).

Estimates the local standard deviation using Gaussian Kernel Smoothing.
